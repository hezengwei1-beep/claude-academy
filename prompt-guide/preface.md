# 《Claude Prompt 工程》

## 为什么专门讲 Claude 的 Prompt

通用 prompt 技巧让你"能用"，Claude 专属技巧让你"用得好"。

Claude 有独特的特性：XML 标签亲和力、Extended Thinking、Prompt Caching、对 System Prompt 的高遵循度。本书专注这些差异化能力。
