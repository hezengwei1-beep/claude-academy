
# 第四章 20 分钟上手 Cowork

本章目标：从零开始，20 分钟内完成 Cowork 的安装、配置，并成功运行第一个任务。

## 前置条件

在开始之前，确认你满足以下条件：

| 条件 | 要求 |
|------|------|
| Claude 付费计划 | Pro（$20/月）、Max、Team 或 Enterprise |
| 操作系统 | macOS（Apple Silicon：M1/M2/M3/M4）或 Windows |
| 应用 | Claude Desktop（最新版） |
| 网络 | 稳定的互联网连接 |

> **注意**：Intel Mac 不支持 Cowork。如果你使用 Intel Mac，请直接跳到 Part 3 使用 Claude Code。

## 第一步：安装 Claude Desktop（2 分钟）

1. 访问 claude.com/download
2. 下载对应你操作系统的安装包
3. 安装并启动 Claude Desktop
4. 用你的 Claude 账号登录

如果你已经安装了 Claude Desktop，确保更新到最新版本。Cowork 功能需要较新的版本才能使用。

## 第二步：切换到 Cowork 模式（1 分钟）

打开 Claude Desktop 后，你会看到顶部有两个标签：

- **Chat**：传统的聊天模式（一问一答）
- **Cowork**：任务模式（Claude 自主执行多步骤任务）

点击 **Cowork** 标签，你就进入了任务模式。

两者的核心区别：
- Chat = 你问一句，Claude 答一句
- Cowork = 你描述一个任务，Claude 制定计划并自主执行

## 第三步：设置 Global Instructions（5 分钟）

这是最重要的一步，但很多新手跳过了它。Global Instructions 相当于给 Claude 一份"关于你"的说明书，让它在每次对话中都了解你的背景和偏好。

**操作路径**：Settings > Cowork > Global Instructions

**推荐模板**：

```
## 关于我
- 角色：[你的职业/角色]
- 行业：[你所在的行业]
- 主要使用语言：中文

## 工作偏好
- 输出格式：[Markdown / 纯文本 / 表格优先]
- 语气：[专业正式 / 简洁直接 / 轻松友好]
- 文件命名：[你的命名习惯]

## 常用工具和文件
- 常用文件夹：[路径]
- 输出目录：[路径]
```

**为什么这一步重要**：没有 Global Instructions，每次新对话 Claude 都从零开始了解你。有了它，Claude 从第一句话就知道你是谁、你想要什么风格的输出。

## 第四步：创建工作文件夹（3 分钟）

给 Claude 一个专门的"工作空间"，而不是让它访问你的整个文件系统。

**推荐结构**：

```
~/Claude Cowork/
├── ABOUT ME/          # 放你的个人资料、偏好文档
│   ├── 工作简介.md
│   └── 常用模板.md
├── INPUTS/            # 放需要 Claude 处理的原始材料
│   ├── 会议记录/
│   └── 待整理文件/
├── OUTPUTS/           # Claude 的输出会放在这里
│   ├── 报告/
│   └── 整理后的文件/
└── TEMPLATES/         # 你的模板文件
    ├── 周报模板.md
    └── 会议纪要模板.md
```

## 第五步：授予文件权限（2 分钟）

在 Cowork 中，你需要明确告诉 Claude 它可以访问哪些文件夹。

**操作方式**：在 Cowork 界面中选择工作文件夹时，选择你刚创建的 `~/Claude Cowork/` 目录。

> **安全提醒**：切勿授予整个 Home 目录的权限。只授予 Claude 需要操作的特定文件夹。这既是安全最佳实践，也能避免 Claude 被无关文件分散注意力。

## 第六步：运行第一个任务（5 分钟）

现在，让我们来一个简单但实用的任务——整理 Downloads 文件夹。

**在 Cowork 输入框中输入**：

```
请帮我整理 ~/Downloads 文件夹：
1. 按文件类型分类（PDF、图片、文档、压缩包、其他）
2. 每个类别创建一个子文件夹
3. 将文件移动到对应的子文件夹
4. 生成一份整理报告，列出移动了哪些文件

注意：不要删除任何文件，只做移动和分类。
```

**Claude 会做什么**：
1. 扫描 Downloads 文件夹中的所有文件
2. 分析每个文件的类型
3. 创建分类子文件夹
4. 逐个移动文件
5. 生成整理报告

**你需要做什么**：
1. 审查 Claude 的执行计划（它会先展示计划再执行）
2. 确认后让它执行
3. 检查结果

恭喜！你已经完成了 Cowork 的首次使用。

## 新手五大常见错误

### 错误一：没设 Global Instructions

**症状**：每次新对话都要花前 5 分钟重复介绍自己。

**解决**：花 5 分钟设置一次 Global Instructions，以后每次都省时间。

### 错误二：把 Cowork 当聊天机器人用

**症状**：一问一答，问"这个文件是什么？"而不是"帮我整理这些文件"。

**解决**：Cowork 的价值在于委派完整任务。给它目标和约束，让它自主执行。好的指令 = 明确的目标 + 清晰的约束 + 预期的交付物。

### 错误三：授予太大的文件权限

**症状**：Claude 扫描了大量无关文件，运行变慢，甚至误操作。

**解决**：只授予必要的文件夹权限。用专门的工作目录，不要给整个 Home。

### 错误四：不看执行计划就让它跑

**症状**：Claude 做了你不想要的操作。

**解决**：Cowork 在执行前会展示计划。花 30 秒看一下计划，确认没有问题再让它执行。

### 错误五：任务描述太模糊

**症状**：结果不符合预期。

**解决**：好的任务描述包含三要素：
1. **目标**：你想要什么结果
2. **约束**：不能做什么、边界在哪里
3. **交付物**：期望的输出格式和位置

**模糊示例**："帮我整理文件"

**清晰示例**："帮我整理 ~/Downloads 中过去 30 天的文件，按类型分到 PDF/图片/文档 三个子文件夹，不要删除任何文件，完成后生成 Markdown 格式的整理报告放到 ~/Claude Cowork/OUTPUTS/"
