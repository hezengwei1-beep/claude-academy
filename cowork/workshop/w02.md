
# Workshop 2：用 Cowork 搭建日常自动化工作流

**预计耗时**：20-30 分钟
**前置条件**：Claude Desktop 已安装，Pro 及以上付费计划
**目标**：创建 3 个可复用的 Skills，形成每日自动化工作流

## 准备工作（5 分钟）

### 1. 创建工作目录

在你的电脑上创建以下文件夹结构：

```
~/Claude Cowork/
├── ABOUT ME/
│   └── 个人简介.md
├── INPUTS/
│   ├── 今日笔记/
│   └── 待整理/
├── OUTPUTS/
├── TEMPLATES/
│   ├── 日报模板.md
│   └── 周报模板.md
└── SKILLS/
```

### 2. 写个人简介

在 `ABOUT ME/个人简介.md` 中写：

```markdown
## 关于我
- 职业：[你的职业]
- 主要工作：[日常做什么]
- 输出偏好：Markdown 格式，中文，简洁直接
```

### 3. 创建日报模板

在 `TEMPLATES/日报模板.md` 中写：

```markdown
# 日报 - {日期}

## 今日完成
- 

## 进行中
- 

## 明日计划
- 

## 备注
```

## 动手练习一：文件整理 Skill（10 分钟）

### Step 1：打开 Cowork

Claude Desktop → Cowork 标签 → 选择 `~/Claude Cowork/` 作为工作文件夹

### Step 2：试运行

输入以下任务：

```
请整理 ~/Downloads 文件夹中最近 7 天下载的文件：
1. 按类型分类：PDF、图片、文档、其他
2. 对 PDF 文件读取内容判断是发票、文档还是电子书
3. 不要移动文件，只生成分类报告
4. 报告输出到 ~/Claude Cowork/OUTPUTS/文件分类报告.md

注意：这次只做报告，不要移动任何文件。
```

### Step 3：检查结果

打开 `OUTPUTS/文件分类报告.md`，看分类是否合理。

### Step 4：保存为 Skill

确认效果后，在 Cowork Settings 中创建 Skill：
- 名称：`文件整理报告`
- 指令：把上面的提示词粘贴进去
- 以后只需输入"运行文件整理报告"即可

## 动手练习二：日报生成 Skill（10 分钟）

### Step 1：准备输入

在 `INPUTS/今日笔记/` 下创建几个简单的笔记文件（可以是今天的工作记录、会议要点等）。

### Step 2：运行

```
基于 ~/Claude Cowork/INPUTS/今日笔记/ 下的所有文件，
使用 ~/Claude Cowork/TEMPLATES/日报模板.md 的格式，
生成今天的日报。

要求：
1. 从笔记中提取完成事项、进行中事项
2. 基于进行中的事项推导明日计划
3. 输出到 ~/Claude Cowork/OUTPUTS/日报-{今天日期}.md
```

### Step 3：审查和保存为 Skill

检查输出 → 调整提示词 → 保存为"日报生成" Skill

## 动手练习三：Dispatch 远程测试（5 分钟）

如果你有 Pro/Max 计划且手机已安装 Claude App：

### Step 1：确保桌面端就绪

- Claude Desktop 保持打开
- 电脑不休眠

### Step 2：在手机上发送任务

打开 Claude 手机 App → Dispatch → 输入：

```
运行日报生成 Skill
```

### Step 3：在手机上查看结果

等待桌面端执行完成，在手机上查看生成的日报。

## 复盘

完成以上练习后，你已经拥有了：

- [ ] 一个文件整理报告 Skill（可随时运行）
- [ ] 一个日报生成 Skill（每天下班前运行）
- [ ] Dispatch 远程触发的能力（可选）

**日常工作流建议**：

```
早上：检查昨天的输出，规划今天的笔记
工作中：随手把笔记放到 INPUTS/今日笔记/
下班前：运行"日报生成" Skill（2 分钟）
每周五：把本周日报合成周报（可以再做一个 Skill）
```

## 进阶挑战

1. 创建一个"周报汇总" Skill，自动基于本周所有日报生成周报
2. 创建一个"邮件起草" Skill，基于日报为上级起草进度更新邮件
3. 用 Dispatch 在通勤路上触发晚间整理任务
