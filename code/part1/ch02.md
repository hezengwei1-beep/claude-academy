# 第二章 核心命令与斜杠命令

## 内置斜杠命令

Claude Code 内置了一系列 `/` 开头的命令：

### 对话管理

| 命令 | 作用 |
|------|------|
| `/clear` | 清除当前对话历史 |
| `/compact` | 压缩对话历史（保留关键信息，释放上下文） |
| `/resume` | 恢复之前的会话 |
| `/rewind` | 回滚到上一个检查点 |

### 模式切换

| 命令 | 作用 |
|------|------|
| `/plan` | 进入规划模式（Claude 只分析不执行） |
| `/fast` | 切换快速模式（同模型，更快输出） |

### 工具管理

| 命令 | 作用 |
|------|------|
| `/mcp` | 管理 MCP Servers |
| `/agents` | 管理子代理 |
| `/loop` | 设置定时循环任务 |

### 其他

| 命令 | 作用 |
|------|------|
| `/help` | 显示帮助 |
| `/exit` | 退出 |
| `/commit` | 提交代码（如果有对应 Skill） |

## 键盘快捷键

| 快捷键 | 作用 |
|--------|------|
| `Enter` | 发送消息 |
| `Shift+Enter` | 换行（不发送） |
| `Ctrl+C` | 中断当前操作 |
| `Ctrl+D` | 退出 |
| `Esc` | 取消当前输入 |
| `Up/Down` | 浏览历史输入 |
| `Tab` | 自动补全文件路径 |
| `Shift+Tab` | 切换 Delegate Mode（Agent Teams） |

## CLI 参数

```bash
# 基础用法
claude                           # 交互模式
claude -p "任务描述"             # 非交互模式
claude --resume                  # 恢复上次会话

# 配置
claude --model sonnet            # 指定模型
claude --worktree feature-x      # 在 worktree 中启动

# 输出控制
claude -p "任务" --output json   # JSON 格式输出
claude -p "任务" --verbose       # 详细输出
```

## 实用技巧

### 文件引用

在对话中直接引用文件：

```
看一下 src/auth/middleware.ts 有什么问题
```

Claude 会自动读取这个文件。

### 图片输入

Claude Code 可以读取图片：

```
看一下 screenshot.png 里的错误信息，帮我分析
```

### 多行输入

按 `Shift+Enter` 换行，适合输入多行指令或代码块：

```
帮我写一个函数：
- 输入：用户邮箱
- 输出：是否是有效格式
- 用正则表达式
- 包含单元测试
```

### ! 前缀运行 shell

在对话中直接运行 shell 命令：

```
! ls -la src/
! git log --oneline -5
! npm test
```

输出会直接显示在对话中，Claude 也能看到。
