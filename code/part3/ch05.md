# 第五章 Settings 深度配置

## 配置文件层级

Claude Code 的配置从四个层级加载，优先级从高到低：

```
1. 命令行参数         → 最高优先级
2. 项目级 .claude/    → 项目特有配置
3. 用户级 ~/.claude/  → 个人全局配置
4. 默认值             → 兜底
```

## settings.json

主配置文件位于 `~/.claude/settings.json`：

```json
{
  "model": "claude-sonnet-4-6",
  "env": {
    "CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS": "1"
  },
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "~/Documents"]
    }
  },
  "permissions": {
    "allow": [
      "Read",
      "Glob",
      "Grep"
    ],
    "deny": []
  },
  "teammateMode": "in-process"
}
```

### 常用配置项

| 字段 | 作用 | 示例值 |
|------|------|--------|
| `model` | 默认模型 | `"claude-opus-4-6"` / `"claude-sonnet-4-6"` |
| `env` | 环境变量 | 启用实验性功能 |
| `mcpServers` | MCP Server 配置 | 见 MCP 全书 |
| `permissions` | 权限策略 | allow/deny 工具列表 |
| `teammateMode` | Agent Teams 显示 | `"in-process"` / `"tmux"` |

## 权限管理

### 权限模式

Claude Code 有三种权限模式：

| 模式 | 行为 |
|------|------|
| **Ask**（默认） | 危险操作前询问 |
| **Auto-allow** | 匹配的操作自动允许 |
| **Deny** | 匹配的操作自动拒绝 |

### 配置示例

```json
{
  "permissions": {
    "allow": [
      "Read",
      "Glob",
      "Grep",
      "Bash(npm test)",
      "Bash(npm run lint)"
    ],
    "deny": [
      "Bash(rm -rf *)",
      "Bash(git push --force)"
    ]
  }
}
```

这个配置：
- 自动允许读取、搜索
- 自动允许 `npm test` 和 `npm run lint`
- 自动拒绝 `rm -rf` 和 `git push --force`
- 其他操作每次询问

### 权限设计原则

1. **读操作自动允许**：读取文件、搜索代码没有风险
2. **测试命令自动允许**：跑测试不会改变代码
3. **写操作逐次确认**：修改文件需要你看一眼
4. **危险操作禁止**：force push、rm -rf 等永远不自动允许

## 项目级配置

在项目根目录创建 `.claude/` 目录：

```
my-project/
├── .claude/
│   ├── settings.json    → 项目级设置（可提交到 Git）
│   └── settings.local.json → 个人设置（不提交到 Git）
├── .mcp.json            → MCP Server 配置（项目级）
├── CLAUDE.md            → 项目级指令
└── src/
```

### 团队共享 vs 个人

| 文件 | 提交到 Git？ | 内容 |
|------|-------------|------|
| `.claude/settings.json` | 是 | 团队共享的项目配置 |
| `.claude/settings.local.json` | 否 | 个人偏好覆盖 |
| `.mcp.json` | 是 | 团队共享的 MCP Server |
| `CLAUDE.md` | 是 | 项目约定 |
| `~/.claude/CLAUDE.md` | N/A | 个人全局偏好 |
