## 效率翻倍的习惯

### 1. 给上下文而不是指令

```
# 差
修复 bug

# 好
用户反馈登录偶尔失败，错误日志显示 JWT 验证超时。
相关代码在 src/auth/verify.ts。帮我排查。
```

### 2. 先问再改

```
# 差
直接重构这个模块

# 好
分析 src/legacy/processor.ts 的问题，
给出重构方案让我确认后再动手
```

### 3. 分步执行

```
# 差
重写整个认证系统

# 好
第一步：分析当前认证系统的架构
（确认后）
第二步：设计新的 JWT 认证方案
（确认后）
第三步：实现 JWT 工具函数
...
```

### 4. 利用 /compact

长对话后上下文会膨胀。定期用 `/compact` 压缩：

```
/compact
```

Claude 会保留关键信息，释放上下文空间。

### 5. 利用 /resume

中断后下次继续：

```bash
claude --resume
```

恢复之前的对话上下文，不用重新解释背景。
