# 前言

## 为什么需要这本书

Claude Code 是 Anthropic 推出的 CLI 编码代理——你在终端里跟它说话，它直接帮你读代码、改代码、跑命令。

听起来简单，但大多数人只用了它 20% 的能力。

这本书覆盖剩下的 80%：从安装配置到 Hooks 自动化、从权限管理到 IDE 集成、从单次对话到持久化工作流。它是 Claude 学院系列的**入口级教材**——其他三本书（Prompt 实战、协作全书、MCP 全书）都假设你会用 Claude Code，这本书确保你真的会。

## 结构

| Part | 内容 | 面向 |
