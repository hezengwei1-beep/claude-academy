# 前言

## 为什么需要这本书

Claude Code 是 Anthropic 推出的 CLI 编码代理——你在终端里跟它说话，它直接帮你读代码、改代码、跑命令。

听起来简单，但大多数人只用了它 20% 的能力。

这本书覆盖剩下的 80%：从安装配置到 Hooks 自动化、从权限管理到 IDE 集成、从单次对话到持久化工作流。它是 Claude 学院系列的**入口级教材**——其他五本书都假设你会用 Claude Code，这本书确保你真的会。

## 这本书解决什么问题

| 你的现状 | 读完后 |
|----------|--------|
| 只会 `claude` 启动，然后聊天 | 掌握 `/plan`、`/compact`、`/resume` 等 20+ 命令 |
| 每次操作都要点"允���" | 配置好权限策略，安全操作自动允许 |
| 不知道 Hooks 是什么 | 文件修改后自动 lint + format + test |
| 只在终端用 | VS Code / JetBrains 无缝集成 |
| 不敢在 CI 里用 | GitHub Actions 自动代码审查 |
| 觉得"变慢了"不知道为什么 | 掌握上下文管理和成本优化 |

## 前置条件

- 一台 Mac（macOS 14+）或 Linux 机器
- Node.js >= 18
- 一个 Anthropic 账号（免费注册，API 按量计费）
- 基本的终端使用经验
