
# 前言

## 为什么写这本书

Claude 很聪明，但它是一个"没有手脚的大脑"——它能思考，但无法直接操作你的 GitHub、查你的数据库、发你的 Slack 消息。

MCP（Model Context Protocol）给了它手脚。

通过 MCP，Claude 可以连接到几乎任何外部系统——数据库、API、SaaS 工具、本地文件系统。而且不是"教它用每一个 API"，而是一个标准协议，装一个 Server 就多一种能力。

但 MCP 的文档虽然完整，却分散在协议规范、SDK 文档、GitHub 仓库等多个地方。对于想从零开始的人来说，入口不太友好。

这本书把所有你需要知道的信息整合在一起——从"什么是 MCP"到"用 Python 写一个 Server"到"企业级安全部署"。
