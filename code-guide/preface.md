# 《Claude Code 完全指南》

## 这本书覆盖什么

从 `npm install` 到 20 个生产级工作流——一本书搞定 Claude Code 和 Cowork 的所有用法。

## 结构

| Part | 内容 | 章节 | 面向 |
|------|------|------|------|
| **Part 1** | 快速入门 | Ch1-2 | 安装、命令、快捷键 |
| **Part 2** | 日常高效 | Ch3-4 | 编码用法、对话管理、上下文控制 |
| **Part 3** | Cowork 桌面 | Ch5-9 | 上手、十大场景、Dispatch、Computer Use、Skills |
| **Part 4** | 深度配置 | Ch10-15 | Settings、权限、Hooks、Slash Commands、Memory |
| **Part 5** | 团队与自动化 | Ch16-18 | IDE 集成、CI/CD、Managed Agents |
| **Part 6** | 20 个 Recipes | Ch19-22 | Git、文档、分析、大型工程——完整工作流 |
| **附录** | 速查 | A-C | 命令表、性能调优、排障 |

## 与其他书的关系

```
《Code 完全指南》 ← 你在这里（工具使用）
    ↓
《Prompt 工程》   → 写好 prompt
    ↓
《Agent 完全指南》→ 理解和构建 Agent
    ↓
《MCP 完全指南》  → 连接外部世界
    ↓
《企业实战》      → 案例、成本、安全
```
