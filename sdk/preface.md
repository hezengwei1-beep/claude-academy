# 前言

## 从使用者到造物者

前五本书教你**使用** Claude 的各种能力。这本书教你**用代码构建**自己的 AI 代理系统。

Agent SDK 让你可以：
- 把 Claude 嵌入你的 Web 应用
- 构建自定义的 CLI 工具
- 搭建 Slack/Discord Bot
- 在 CI/CD 中编排多代理流水线
- 构建产品级的 AI 功能

## 前置条件

- TypeScript 或 Python 开发经验
- 了解 Claude API 基础（或阅读过《Prompt 工程实战》）
- 有实际的集成需求

## 结构

| Part | 内容 |
|------|------|
| Part 1 | SDK 基础：安装、第一个 Agent、自定义工具 |
| Part 2 | 多代理编排：SubAgent、并行、通信 |
| Part 3 | 产品集成：Web 应用、CLI、Bot、CI/CD |
| Part 4 | 生产运维：评估、监控、成本、安全 |
