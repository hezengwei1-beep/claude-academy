# 前言

## 这本书不讲概念

前面四本书教你 Claude Code **是什么、怎么用、怎么配**。这本书只做一件事——给你**可以直接复制使用的完整工作流**。

每个 Recipe 包含：
- **场景**：什么时候用
- **完整配置**：settings.json / hooks / CLAUDE.md
- **完整指令**：复制粘贴即可执行
- **预期结果**：你会得到什么
- **成本估算**：大概花多少钱

## 结构

| Part | 内容 |
|------|------|
| Part 1 | 自定义扩展（Slash Commands、Keybindings、Memory） |
| Part 2 | 远程与自动化（Managed Agents、Headless、Triggers） |
| Part 3 | 20 个实战 Recipes |
| Part 4 | 企业部署 |

## 前置条件

- 已阅读《Claude Code 从零精通》或有同等使用经验
- Claude Code >= v2.1.32
- 付费 API 账户
