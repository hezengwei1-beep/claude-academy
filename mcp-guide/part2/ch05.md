
# 第五章 社区 MCP Servers 精选

社区 MCP Server 生态非常活跃，`awesome-mcp-servers` 项目收录了数百个 Server。本章精选最实用的分类介绍。

## 数据库类

### PostgreSQL / MySQL / SQLite

让 Claude 直接查询你的数据库。

```json
{
  "postgres": {
    "command": "npx",
    "args": ["-y", "@mcp/server-postgres"],
    "env": {
      "DATABASE_URL": "postgresql://user:pass@localhost:5432/mydb"
    }
  }
}
```

**典型用途**：
- "查一下上个月注册的用户数"
- "分析 orders 表中金额最大的 10 笔订单"
- "帮我写一个查询，找出 7 天未登录的活跃用户"

**安全提示**：建议使用只读数据库用户，避免 Claude 执行写入操作。

### MongoDB / Supabase

类似模式，连接 NoSQL 或 BaaS 平台。

## 开发工具类

### GitHub Server

```json
{
  "github": {
    "command": "npx",
    "args": ["-y", "@modelcontextprotocol/server-github"],
    "env": {
      "GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_..."
    }
  }
}
```

**能力**：创建/查看 Issues、PR 操作、代码搜索、仓库管理

### Docker Server

让 Claude 管理 Docker 容器。

**能力**：列出容器、启停容器、查看日志、执行命令

### Sentry Server

让 Claude 分析错误追踪数据。

**能力**：查看最近的错误、分析错误趋势、获取堆栈跟踪

## 选择 Server 的建议

1. **先看官方**：如果官方有，优先用官方版本
2. **看 Stars 和更新频率**：活跃维护的比较可靠
3. **看安全性**：敏感数据相关的 Server 要仔细审查代码
4. **按需安装**：不要一口气装 20 个 Server，需要什么装什么
5. **检查权限**：每个 Server 需要什么环境变量/API Key，最小权限原则

## MCP Registry

官方正在建设 MCP Registry（`registry.modelcontextprotocol.io`），提供：
- 统一的 Server 发现机制
- DNS 验证确保命名空间真实性
- REST API 供客户端查询
- 由 Anthropic、GitHub、Microsoft 等支持

未来安装 MCP Server 可能变得像 `npm install` 一样简单。
