
# 第四章 官方 MCP Servers 大全

## 官方参考实现

Anthropic 在 `github.com/modelcontextprotocol/servers` 维护了一组参考实现。以下是当前活跃的官方 Server。

## Filesystem Server —— 文件系统操作

**功能**：安全的文件读写、目录浏览、文件搜索

**安装**：
```json
{
  "filesystem": {
    "command": "npx",
    "args": ["-y", "@modelcontextprotocol/server-filesystem", "/允许访问的目录"]
  }
}
```

**提供的工具**：
| 工具 | 作用 |
|------|------|
| `read_file` | 读取文件内容 |
| `write_file` | 写入文件 |
| `list_directory` | 列出目录内容 |
| `create_directory` | 创建目录 |
| `move_file` | 移动/重命名文件 |
| `search_files` | 搜索文件 |
| `get_file_info` | 获取文件元数据 |

**安全特性**：只能访问 args 中指定的目录，无法越界。

**适用场景**：文件整理、文档处理、项目管理

## Git Server —— Git 仓库操作

**功能**：Git 仓库的读取、搜索、历史查看

**安装**：
```json
{
  "git": {
    "command": "uvx",
    "args": ["mcp-server-git", "--repository", "/path/to/repo"]
  }
}
```

**提供的工具**：
| 工具 | 作用 |
|------|------|
| `git_log` | 查看提交历史 |
| `git_diff` | 查看文件差异 |
| `git_status` | 查看仓库状态 |
| `git_show` | 查看特定提交内容 |
| `git_blame` | 查看文件每行的最后修改者 |

**适用场景**：代码审查、提交历史分析、变更追踪

```json
{
```

```json
    "args": ["-y", "@modelcontextprotocol/server-memory"]
  }

```

**提供的工具**：
| 工具 | 作用 |
|------|------|

| `search_nodes` | 搜索知识图谱 |

**适用场景**：跨会话记忆、项目知识管理、个人偏好存储

## 组合使用

你可以同时启用多个 Server：

```json
{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "~/Documents"]
    },
    "memory": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-memory"]
    },
    "fetch": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-fetch"]
    }
  }
}
```

这样 Claude 就同时拥有了文件操作、持久记忆、网页获取三种能力。
