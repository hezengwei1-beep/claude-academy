# 第六章 第三方 Agent 框架与 Claude 集成

## 框架全景

| 框架 | 语言 | 定位 | 与 Claude 集成 |
|------|------|------|---------------|
| **LangGraph** | Python/JS | 有状态的多 Agent 图编排 | ✅ 原生支持 Claude |
| **CrewAI** | Python | 角色驱动的多 Agent 协作 | ✅ 支持 Claude |
| **AutoGen** | Python | 微软的多 Agent 对话框架 | ✅ 支持 Claude |
| **LangChain** | Python/JS | 通用 LLM 应用框架 | ✅ 原生支持 |
| **Anthropic SDK** | Python/JS | 官方 API SDK | ✅ 原生 |
| **Agent SDK** | JS | Claude Code Agent 封装 | ✅ 原生 |

## 何时用 Claude 原生 vs 第三方框架

| 场景 | 推荐 |
|------|------|
| 编码任务（改代码、跑测试） | Claude Code / Agent SDK |
| 复杂的多步骤工作流 | LangGraph |
| 角色扮演式多 Agent | CrewAI |
| 研究和实验 | AutoGen |
| 简单的 API 调用 | Anthropic SDK |
| 产品集成 | Agent SDK 或 LangChain |

**经验法则**：如果任务涉及代码和文件操作，用 Claude 原生方案（Agent SDK）。如果任务是通用的 AI 编排（不限于编码），考虑第三方框架。

## LangGraph + Claude

LangGraph 是 LangChain 团队的图编排框架，特别适合复杂的有状态工作流。

### 安装

```bash
pip install langgraph langchain-anthropic
```

### 基础示例：带工具的 Agent

```python
from langgraph.prebuilt import create_react_agent
from langchain_anthropic import ChatAnthropic
from langchain_core.tools import tool

# Claude 模型
model = ChatAnthropic(model="claude-sonnet-4-6")

# 定义工具
@tool
def search_code(query: str) -> str:
    """在代码库中搜索关键词"""
    import subprocess
    result = subprocess.run(
        ["grep", "-rn", query, "src/"],
        capture_output=True, text=True
    )
    return result.stdout[:2000]  # 限制返回长度

@tool
def run_tests() -> str:
    """运行项目测试"""
    import subprocess
    result = subprocess.run(
        ["npm", "test"], capture_output=True, text=True, timeout=60
    )
    return f"stdout:\n{result.stdout[:1000]}\nstderr:\n{result.stderr[:1000]}"

# 创建 Agent
agent = create_react_agent(model, tools=[search_code, run_tests])

# 运行
result = agent.invoke({
    "messages": [{"role": "user", "content": "检查项目中的 TODO 并运行测试"}]
})
print(result["messages"][-1].content)
```

### 进阶：多 Agent 图

```python
from langgraph.graph import StateGraph, MessagesState

# 定义状态
class ReviewState(MessagesState):
    security_report: str = ""
    quality_report: str = ""
    final_report: str = ""

# 定义节点（每个节点是一个 Agent）
async def security_review(state: ReviewState):
    model = ChatAnthropic(model="claude-sonnet-4-6")
    response = await model.ainvoke([
        {"role": "system", "content": "你是安全审查专家"},
        {"role": "user", "content": "审查安全性"}
    ])
    return {"security_report": response.content}

async def quality_review(state: ReviewState):
    model = ChatAnthropic(model="claude-sonnet-4-6")
    response = await model.ainvoke([
        {"role": "system", "content": "你是代码质量专家"},
        {"role": "user", "content": "审查代码质量"}
    ])
    return {"quality_report": response.content}

async def synthesize(state: ReviewState):
    model = ChatAnthropic(model="claude-sonnet-4-6")
    response = await model.ainvoke([
        {"role": "user", "content": f"""综合以下报告：
安全：{state['security_report']}
质量：{state['quality_report']}
生成最终评估。"""}
    ])
    return {"final_report": response.content}

# 构建图
graph = StateGraph(ReviewState)
graph.add_node("security", security_review)
graph.add_node("quality", quality_review)
graph.add_node("synthesize", synthesize)

# 安全和质量并行，然后综合
graph.add_edge("__start__", "security")
graph.add_edge("__start__", "quality")
graph.add_edge("security", "synthesize")
graph.add_edge("quality", "synthesize")
graph.add_edge("synthesize", "__end__")

app = graph.compile()
result = await app.ainvoke({"messages": []})
print(result["final_report"])
```

## CrewAI + Claude

CrewAI 的核心理念：给每个 Agent 一个"角色"，让它们像真实团队一样协作。

### 安装

```bash
pip install crewai crewai-tools
```

### 示例

```python
from crewai import Agent, Task, Crew
from langchain_anthropic import ChatAnthropic

llm = ChatAnthropic(model="claude-sonnet-4-6")

# 定义角色
researcher = Agent(
    role="技术调研员",
    goal="调研技术方案的优劣",
    backstory="你是一位有 10 年经验的技术架构师",
    llm=llm,
    verbose=True,
)

writer = Agent(
    role="技术文档作者",
    goal="将调研结果整理成清晰的技术文档",
    backstory="你是一位专业的技术写作者",
    llm=llm,
    verbose=True,
)

# 定义任务
research_task = Task(
    description="调研 Redis vs Memcached 做缓存层的优劣",
    expected_output="详细的对比分析",
    agent=researcher,
)

write_task = Task(
    description="基于调研结果写一份技术选型文档",
    expected_output="Markdown 格式的技术选型文档",
    agent=writer,
)

# 组建团队
crew = Crew(
    agents=[researcher, writer],
    tasks=[research_task, write_task],
    verbose=True,
)

result = crew.kickoff()
print(result)
```

## 框架选择决策

```
你想做什么？
│
├─ 代码相关（读/写/测试） → Agent SDK
│
├─ 有状态的复杂工作流 → LangGraph
│   （需要条件分支、循环、并行）
│
├─ 角色驱动的团队协作 → CrewAI
│   （每个 Agent 有明确的角色和职责）
│
├─ 简单的 API 调用 → Anthropic SDK
│   （不需要 Agent 循环）
│
└─ 通用的 LLM 应用 → LangChain
    （RAG、Chain、简单工具调用）
```

## 与 Claude Code 的关系

这些框架和 Claude Code 不是竞争关系，而是不同层次：

```
Claude Code（终端交互）
  → 开发者日常使用
  → Agent Teams 协作

Agent SDK（编程接口）
  → 嵌入产品
  → CI/CD 自动化

第三方框架（通用编排）
  → 非编码场景的复杂工作流
  → 跨模型的 Agent 系统
  → 研究和实验
```

大多数开发者只需要 Claude Code + Agent SDK 就够了。第三方框架在以下情况值得考虑：
- 需要跨厂商模型（同时用 Claude 和 GPT）
- 需要复杂的状态机（LangGraph 更擅长）
- 团队已经在用 LangChain 生态
