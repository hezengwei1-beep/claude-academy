# 第五章 从零构建你的第一个 Agent

## 目标

用 Agent SDK 构建一个"代码健康检查 Agent"——输入一个项目路径，自动分析代码质量并生成报告。

## 项目初始化

```bash
mkdir health-check-agent && cd health-check-agent
npm init -y
npm install @anthropic-ai/claude-code
npm install -D typescript @types/node
```

`tsconfig.json`：
```json
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "Node16",
    "moduleResolution": "Node16",
    "outDir": "./dist",
    "rootDir": "./src",
    "strict": true
  }
}
```

`package.json` 添加：
```json
{
  "type": "module",
  "scripts": {
    "build": "tsc",
    "start": "node dist/index.js",
    "dev": "tsc && node dist/index.js"
  }
}
```

## 第一版：最简单的 Agent

```typescript
// src/index.ts
import { createAgent } from '@anthropic-ai/claude-code';

const projectPath = process.argv[2] || '.';

const agent = createAgent({
  model: 'claude-sonnet-4-6',
  systemPrompt: `你是一个代码健康检查专家。
分析项目的代码质量，检查以下维度：
1. 项目结构是否合理
2. 依赖是否有安全漏洞
3. 代码中是否有明显的 bad practice
4. 测试覆盖情况

输出一份结构化的健康报告。`,
  tools: ['Read', 'Glob', 'Grep', 'Bash'],
  workingDirectory: projectPath,
  maxTokens: 200000,
  timeout: 120000,
});

console.log(`正在分析项目: ${projectPath}`);
const result = await agent.run('对这个项目做一次全面的代码健康检查');
console.log(result.output);
```

### 运行

```bash
npm run dev -- ~/Projects/my-app
```

**这就是一个完整的 Agent**。7 行配置 + 1 行运行 = 一个能自主分析代码质量的 Agent。

## 第二版：添加自定义工具

让 Agent 能查依赖漏洞和统计代码行数：

```typescript
// src/tools.ts
import { Tool } from '@anthropic-ai/claude-code';
import { execSync } from 'child_process';

export const checkDependencies: Tool = {
  name: 'check_dependencies',
  description: '检查项目的 npm 依赖安全漏洞和过期状态',
  parameters: {
    type: 'object',
    properties: {
      checkType: {
        type: 'string',
        enum: ['audit', 'outdated', 'both'],
        description: 'audit=安全漏洞, outdated=过期包, both=两者都查',
      },
    },
    required: ['checkType'],
  },
  execute: async ({ checkType }) => {
    const results: string[] = [];

    if (checkType === 'audit' || checkType === 'both') {
      try {
        const audit = execSync('npm audit --json 2>/dev/null', { encoding: 'utf8' });
        const parsed = JSON.parse(audit);
        results.push(`安全漏洞: ${parsed.metadata?.vulnerabilities?.total || 0} 个`);
      } catch {
        results.push('安全漏洞: 无法检查（可能没有 package-lock.json）');
      }
    }

    if (checkType === 'outdated' || checkType === 'both') {
      try {
        const outdated = execSync('npm outdated --json 2>/dev/null', { encoding: 'utf8' });
        const parsed = JSON.parse(outdated || '{}');
        results.push(`过期包: ${Object.keys(parsed).length} 个`);
      } catch {
        results.push('过期包: 0 个');
      }
    }

    return { data: results.join('\n') };
  },
};

export const countLines: Tool = {
  name: 'count_lines',
  description: '统计项目的代码行数，按文件类型分组',
  parameters: {
    type: 'object',
    properties: {
      directory: {
        type: 'string',
        description: '要统计的目录，默认 src/',
      },
    },
  },
  execute: async ({ directory = 'src' }) => {
    try {
      const result = execSync(
        `find ${directory} -name "*.ts" -o -name "*.tsx" -o -name "*.js" -o -name "*.jsx" | ` +
        `xargs wc -l 2>/dev/null | tail -1`,
        { encoding: 'utf8' }
      );
      return { data: `代码总行数: ${result.trim()}` };
    } catch {
      return { data: '无法统计代码行数' };
    }
  },
};
```

### 使用自定义工具

```typescript
// src/index.ts
import { createAgent } from '@anthropic-ai/claude-code';
import { checkDependencies, countLines } from './tools.js';

const agent = createAgent({
  model: 'claude-sonnet-4-6',
  systemPrompt: `你是代码健康检查专家。
使用所有可用工具全面分析项目。
必须使用 check_dependencies 检查漏洞和过期包。
必须使用 count_lines 统计代码量。
最终输出结构化的健康报告。`,
  tools: ['Read', 'Glob', 'Grep', 'Bash', checkDependencies, countLines],
  workingDirectory: process.argv[2] || '.',
});

const result = await agent.run('执行全面的代码健康检查');
console.log(result.output);
```

## 第三版：多 Agent 协作

用 SubAgent 并行检查不同维度：

```typescript
// src/index.ts
import { createAgent, createSubAgent } from '@anthropic-ai/claude-code';
import { checkDependencies, countLines } from './tools.js';

const projectPath = process.argv[2] || '.';

// 主 Agent（协调者）
const lead = createAgent({
  model: 'claude-sonnet-4-6',
  systemPrompt: '你是项目健康检查的协调者。综合各维度的检查结果，生成最终报告。',
  workingDirectory: projectPath,
});

// 安全检查 SubAgent
const securityChecker = createSubAgent({
  name: 'security',
  model: 'claude-sonnet-4-6',
  systemPrompt: '你是安全专家。只关注安全问题：依赖漏洞、硬编码密钥、注入风险。',
  tools: ['Read', 'Grep', checkDependencies],
  workingDirectory: projectPath,
});

// 代码质量 SubAgent
const qualityChecker = createSubAgent({
  name: 'quality',
  model: 'claude-sonnet-4-6',
  systemPrompt: '你是代码质量专家。检查：复杂度、重复代码、命名规范、类型安全。',
  tools: ['Read', 'Grep', 'Glob', countLines],
  workingDirectory: projectPath,
});

// 测试覆盖 SubAgent
const testChecker = createSubAgent({
  name: 'testing',
  model: 'claude-haiku-4-5',  // 用便宜模型
  systemPrompt: '你是测试专家。检查测试覆盖率、测试质量、缺失的测试。',
  tools: ['Read', 'Glob', 'Bash'],
  workingDirectory: projectPath,
});

// 并行执行
console.log('开始健康检查...');
const [security, quality, testing] = await Promise.all([
  securityChecker.run('检查此项目的安全性').catch(e => ({ output: `安全检查失败: ${e.message}` })),
  qualityChecker.run('分析此项目的代码质量').catch(e => ({ output: `质量检查失败: ${e.message}` })),
  testChecker.run('评估此项目的测试状况').catch(e => ({ output: `测试检查失败: ${e.message}` })),
]);

// 综合报告
const report = await lead.run(`
请综合以下三个维度的检查结果，生成最终的健康报告：

## 安全检查结果
${security.output}

## 代码质量结果
${quality.output}

## 测试覆盖结果
${testing.output}

输出格式：
1. 总体评分（A/B/C/D/F）
2. 关键发现（Top 5）
3. 各维度详情
4. 改进建议（按优先级）
`);

console.log(report.output);
```

```bash
npm run dev -- ~/Projects/my-app
```

## 第四版：添加输出持久化

```typescript
import fs from 'fs';
```
