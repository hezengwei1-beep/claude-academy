# 第二章 自定义工具

## 为什么需要自定义工具

内置工具覆盖了文件操作和 shell 命令。但你可能需要：
- 查询你的数据库
- 调用你的内部 API
- 操作你的 SaaS 服务
- 实现特定的业务逻辑

## 定义自定义工具

```typescript
import { createAgent, Tool } from '@anthropic-ai/claude-code';

// 自定义工具：查询数据库
const queryDatabase: Tool = {
  name: 'query_database',
  description: '执行只读 SQL 查询，返回结果',
  parameters: {
    type: 'object',
    properties: {
      sql: {
        type: 'string',
        description: 'SQL 查询语句（只允许 SELECT）',
      },
      database: {
        type: 'string',
        enum: ['users', 'orders', 'analytics'],
        description: '要查询的数据库',
      },
    },
    required: ['sql', 'database'],
  },
  execute: async ({ sql, database }) => {
    // 安全检查
    if (!sql.trim().toUpperCase().startsWith('SELECT')) {
      return { error: '只允许 SELECT 查询' };
    }

    // 执行查询（用你的数据库客户端）
    const db = getDatabase(database);
    const result = await db.query(sql);
    return { data: JSON.stringify(result.rows, null, 2) };
  },
};

// 使用自定义工具
const agent = createAgent({
  model: 'claude-sonnet-4-6',
  tools: ['Read', 'Grep', queryDatabase],  // 混合内置 + 自定义
  systemPrompt: '你是数据分析师，可以查询数据库回答问题。',
});

const result = await agent.run('上个月新增了多少用户？');
```

## 工具设计最佳实践

### 1. 描述要精确

```typescript
// 差
{ description: '获取数据' }

// 好
{ description: '通过 SQL 查询指定数据库，只支持 SELECT 语句，返回最多 1000 行' }
```

Claude 根据描述决定何时调用工具。描述越精确，调用越准确。

### 2. 参数要有约束

```typescript
parameters: {
  properties: {
    limit: {
      type: 'number',
      minimum: 1,
      maximum: 1000,
      default: 100,
      description: '最多返回行数',
    },
  },
}
```

### 3. 错误要友好

```typescript
execute: async (params) => {
  try {
    const result = await doSomething(params);
    return { data: result };
  } catch (error) {
    return {
      error: `操作失败: ${error.message}。请检查参数是否正确。`,
    };
  }
},
```

返回清晰的错误信息，Claude 会根据错误信息调整下一步行动。

### 4. 安全要前置

```typescript
execute: async ({ sql }) => {
  // 白名单检查
  if (!sql.trim().toUpperCase().startsWith('SELECT')) {
    return { error: '安全限制：只允许 SELECT 查询' };
  }

  // 禁止危险关键词
  const forbidden = ['DROP', 'DELETE', 'UPDATE', 'INSERT', 'ALTER', 'TRUNCATE'];
  if (forbidden.some(f => sql.toUpperCase().includes(f))) {
    return { error: '安全限制：禁止的操作' };
  }

  // 限制结果大小
  if (!sql.toUpperCase().includes('LIMIT')) {
    sql += ' LIMIT 100';
  }

  return await executeQuery(sql);
},
```

## 实用自定义工具示例

### HTTP API 调用

```typescript
const callAPI: Tool = {
  name: 'call_api',
  description: '调用内部 REST API',
  parameters: {
    properties: {
      method: { type: 'string', enum: ['GET', 'POST'] },
      path: { type: 'string', description: 'API 路径，如 /api/users' },
      body: { type: 'object', description: 'POST 请求体（可选）' },
    },
    required: ['method', 'path'],
  },
  execute: async ({ method, path, body }) => {
    const response = await fetch(`${API_BASE}${path}`, {
      method,
      headers: { 'Content-Type': 'application/json', 'Authorization': `Bearer ${API_TOKEN}` },
      body: body ? JSON.stringify(body) : undefined,
    });
    return { status: response.status, data: await response.json() };
  },
};
```

### 发送通知

```typescript
const sendSlack: Tool = {
  name: 'send_slack',
  description: '发送 Slack 消息到指定频道',
  parameters: {
    properties: {
      channel: { type: 'string', description: '频道名，如 #dev' },
      message: { type: 'string', description: '消息内容（支持 Markdown）' },
    },
    required: ['channel', 'message'],
  },
  execute: async ({ channel, message }) => {
    await slack.chat.postMessage({ channel, text: message });
    return { success: true };
  },
};
```
