# 第一章 Agent SDK 架构与安装

## SDK 是什么

Claude Code Agent SDK 是一个 TypeScript/JavaScript 库，让你用代码创建和编排 Claude agents。

与直接调用 Anthropic API 的区别：

| 维度 | Anthropic API | Agent SDK |
|------|--------------|-----------|
| 抽象级别 | 消息级（发消息/收回复） | Agent 级（定义角色/工具/行为） |
| 工具使用 | 手动处理 tool_use 响应 | 自动调用工具并回传结果 |
| 多轮对话 | 手动管理消息历史 | 自动管理 |
| 多 Agent | 需要自己编排 | 内置 SubAgent 支持 |
| 文件操作 | 无 | 内置 Read/Write/Edit 工具 |

## 安装

```bash
npm install @anthropic-ai/claude-code
```

## 第一个 Agent

```typescript
import { createAgent } from '@anthropic-ai/claude-code';

async function main() {
  const agent = createAgent({
    model: 'claude-sonnet-4-6',
    systemPrompt: '你是一个代码助手。简洁回答。',
  });

  const result = await agent.run('用 Python 写一个快速排序函数');
  console.log(result.output);
}

main();
```

## Agent 的生命周期

```
创建 Agent（配置模型、工具、system prompt）
    ↓
运行任务（agent.run(prompt)）
    ↓
Agent 自主决策：
  - 需要读文件？→ 调用 Read 工具
  - 需要搜索？→ 调用 Grep 工具
  - 需要运行命令？→ 调用 Bash 工具
  - 直接回答？→ 生成文本
    ↓
返回结果
```

## Agent 配置项

```typescript
const agent = createAgent({
  // 必需
  model: 'claude-sonnet-4-6',

  // 可选
  systemPrompt: '角色和行为指令',
  tools: ['Read', 'Write', 'Edit', 'Bash', 'Grep', 'Glob'],
  maxTokens: 100000,
  timeout: 300000,       // 5 分钟超时
  workingDirectory: '/path/to/project',

  // 回调
  onToolUse: (tool, params) => {
    console.log(`Using tool: ${tool}`, params);
  },
  onTokenUsage: (usage) => {
    console.log(`Tokens: ${usage.total}`);
  },
});
```

## 内置工具

Agent SDK 复用了 Claude Code 的工具系统：

| 工具 | 作用 |
|------|------|
| `Read` | 读取文件 |
| `Write` | 创建文件 |
| `Edit` | 编辑文件 |
| `Bash` | 执行 shell 命令 |
| `Grep` | 搜索文件内容 |
| `Glob` | 搜索文件名 |
| `Agent` | 创建 SubAgent |

你也可以定义自定义工具（下一章）。

## 工具权限控制

在 Agent 中限制可用工具：

```typescript
// 只读 Agent（安全审查场景）
const readOnlyAgent = createAgent({
  model: 'claude-sonnet-4-6',
  tools: ['Read', 'Grep', 'Glob'],  // 不给 Write/Edit/Bash
  systemPrompt: '你只做代码审查，不要修改任何文件。',
});

// 完整权限 Agent（开发场景）
const devAgent = createAgent({
  model: 'claude-sonnet-4-6',
  tools: ['Read', 'Write', 'Edit', 'Bash', 'Grep', 'Glob'],
});

// 带自定义工具的 Agent
const analyticsAgent = createAgent({
  model: 'claude-sonnet-4-6',
  tools: ['Read', 'Grep', queryDatabase, sendSlack],
});
```

## Streaming 输出

实时流式输出 Agent 的响应：

```typescript
const agent = createAgent({
  model: 'claude-sonnet-4-6',
  onOutput: (chunk) => {
    process.stdout.write(chunk);  // 实时打印
  },
});

await agent.run('分析这个代码库的架构');
```

## 错误处理

```typescript
try {
  const result = await agent.run('修复这个 bug');
  
  if (result.exitCode !== 0) {
    console.error('Agent 任务失败:', result.error);
  }
} catch (error) {
  if (error.code === 'RATE_LIMIT') {
    // API 限流，等待后重试
    await sleep(5000);
    // 重试...
  } else if (error.code === 'CONTEXT_OVERFLOW') {
    // 上下文溢出，需要减少输入
    console.error('输入太大，请分段处理');
  } else if (error.code === 'TIMEOUT') {
    // 超时
    console.error('任务超时，考虑增加 timeout 或拆分任务');
  } else {
    throw error;  // 未知错误
  }
}
```

## 与 Anthropic SDK 的关系

```
Anthropic SDK（@anthropic-ai/sdk）
  └── 底层：消息级 API（messages.create）
  └── 你需要自己处理：tool_use 循环、消息历史、文件操作

Agent SDK（@anthropic-ai/claude-code）
  └── 高层：Agent 级封装
  └── 自动处理：tool_use 循环、消息历史、内置文件/命令工具
  └── 额外能力：SubAgent、并行、工作目录管理
```

**选择**：
- 构建 AI 编码工具 / 自动化脚本 → Agent SDK
- 构建聊天应用 / 内容生成 → Anthropic SDK
- 精细控制每一轮对话 → Anthropic SDK
- 让 Agent 自主完成任务 → Agent SDK
