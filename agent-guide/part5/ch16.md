# 第三章 多代理编排

## SubAgent

SubAgent 是由主 Agent 创建的子代理，完成后向主 Agent 汇报。

```typescript
import { createAgent, createSubAgent } from '@anthropic-ai/claude-code';

const lead = createAgent({
  model: 'claude-opus-4-6',
  systemPrompt: '你是团队负责人，协调下属完成任务。',
});

// 定义 SubAgents
const securityReviewer = createSubAgent({
  name: 'security-reviewer',
  model: 'claude-sonnet-4-6',
  systemPrompt: '你是安全审查专家。只报告 OWASP Top 10 相关问题。',
  tools: ['Read', 'Grep'],
});

const perfReviewer = createSubAgent({
  name: 'perf-reviewer',
  model: 'claude-sonnet-4-6',
  systemPrompt: '你是性能专家。关注查询效率和资源使用。',
  tools: ['Read', 'Grep', 'Bash'],
});
```

## 并行执行

```typescript
// 并行执行多个 SubAgent
const [securityResult, perfResult] = await Promise.all([
  securityReviewer.run('审查 src/auth/ 的安全性'),
  perfReviewer.run('分析 src/auth/ 的性能'),
]);

// 主 Agent 综合
const summary = await lead.run(`
综合以下审查结果：
安全审查：${securityResult.output}
性能审查：${perfResult.output}
生成最终报告。
`);

console.log(summary.output);
```

## 串行链式执行

```typescript
// 第一步：分析
const analysis = await createSubAgent({
  name: 'analyzer',
  model: 'claude-sonnet-4-6',
  tools: ['Read', 'Grep'],
}).run('分析 src/legacy/ 的代码结构和问题');

// 第二步：基于分析结果制定方案
const plan = await createSubAgent({
  name: 'planner',
  model: 'claude-opus-4-6',
}).run(`基于以下分析，制定重构方案：\n${analysis.output}`);

// 第三步：执行方案
const implementation = await createSubAgent({
  name: 'implementer',
  model: 'claude-sonnet-4-6',
  tools: ['Read', 'Write', 'Edit', 'Bash'],
}).run(`按以下方案执行重构：\n${plan.output}`);
```

## 条件分支

```typescript
// 根据第一步结果决定后续策略
const triage = await createSubAgent({
  name: 'triage',
  model: 'claude-haiku-4-5',  // 用最便宜的模型做分类
}).run('这个 bug 报告是前端问题还是后端问题？只回答"前端"或"后端"');

if (triage.output.includes('前端')) {
  await frontendExpert.run('修复前端 bug...');
} else {
  await backendExpert.run('修复后端 bug...');
}
```

## Token Budget 管理

```typescript
const agent = createAgent({
  model: 'claude-sonnet-4-6',
  maxTokens: 500000,

  onTokenUsage: (usage) => {
    const cost = usage.inputTokens * 3 / 1_000_000
                + usage.outputTokens * 15 / 1_000_000;

    if (cost > 5) {
      console.warn(`成本已达 $${cost.toFixed(2)}，考虑停止`);
    }
  },
});
```

## 错误处理与重试

```typescript
async function runWithRetry(agent, prompt, maxRetries = 3) {
  for (let i = 0; i < maxRetries; i++) {
    try {
      return await agent.run(prompt);
    } catch (error) {
      if (error.message.includes('rate_limit') && i < maxRetries - 1) {
        await new Promise(r => setTimeout(r, 2000 * (i + 1)));
        continue;
      }
      throw error;
    }
  }
}
```
